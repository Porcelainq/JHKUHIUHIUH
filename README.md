## Описание программы

Movie Library — это графическое приложение для хранения и управления личной коллекцией фильмов.  
Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга, просматривать их в таблице, фильтровать по жанру и году, а также сохранять все данные в файл JSON для последующей загрузки.

---

## Требования для запуска

- Python 3.10 или выше  
- Стандартная библиотека Tkinter (входит в поставку Python)  
- Дополнительные библиотеки не требуются
